## How It Works

### The Simple Truth

Every service in Backchain Protocol has an `operator` parameter in its functions. When users interact through YOUR interface, YOUR wallet address is passed as the operator, and YOU earn a commission.

```
╔═══════════════════════════════════════════════════════════════════════════════════╗
║                                                                                   ║
║   TRANSACTION FLOW                                                                ║
║   ════════════════                                                                ║
║                                                                                   ║
║   1. You build an interface (website, app, bot, tool)                            ║
║                                                                                   ║
║   2. User accesses YOUR interface                                                ║
║                                                                                   ║
║   3. User performs an action (stake, play, buy, post, etc.)                      ║
║                                                                                   ║
║   4. Your interface calls the smart contract with YOUR address as operator       ║
║      ┌─────────────────────────────────────────────────────────────────────┐     ║
║      │  contract.stake(amount, lockDays, YOUR_WALLET_ADDRESS);             │     ║
║      └─────────────────────────────────────────────────────────────────────┘     ║
║                                                                                   ║
║   5. Smart contract distributes fees:                                            ║
║      - YOUR WALLET receives operator commission (BKC + ETH)                      ║
║      - Stakers receive their share                                               ║
║      - Treasury receives protocol share                                          ║
║                                                                                   ║
║   6. You earned money. The user got the service. Everyone wins.                  ║
║                                                                                   ║
╚═══════════════════════════════════════════════════════════════════════════════════╝
```

### Complete Integration Example

```javascript
import { ethers } from 'ethers';

// YOUR wallet address - this is where commissions go!
const MY_OPERATOR_ADDRESS = '0xYourWalletAddress';

// Contract addresses
const CONTRACTS = {
    DELEGATION: '0x41B1B7940E06318e9b161fc64524FaE7261e8739',
    FORTUNE: '0x5B3c7A322eB49805c594Fd948c137b62a09BBfFF',
    NOTARY: '0x2E56650a4f05D0f98787694c6C61603616716b48',
    BACKCHAT: '0x0D8c2862df03F5be4b569C5ffF4D3aaAEE44BDDb',
    BKC: '0x38DD1898b7B11f07C03e7dBd957fCC8021fB1396'
};

// ABIs (simplified)
const DELEGATION_ABI = [
    "function stake(uint256 amount, uint256 lockDays, address operator) payable",
    "function unstake(uint256 positionId, address operator) payable",
    "function claimRewards(address operator) payable",
    "function getStakeEthFee(uint256 amount) view returns (uint256)"
];

const FORTUNE_ABI = [
    "function commitPlay(uint256 wager, uint256[] guesses, address operator) payable",
    "function revealPlay(address operator) payable returns (uint256[], uint256)",
    "function getCommitEthFee() view returns (uint256)"
];

const NOTARY_ABI = [
    "function notarize(bytes32 hash, address operator) payable returns (uint256)",
    "function getEthFee() view returns (uint256)"
];

const BKC_ABI = [
    "function approve(address spender, uint256 amount) returns (bool)"
];

// ═══════════════════════════════════════════════════════════════════════════════
// STAKING INTERFACE - You earn from every stake/unstake/claim
// ═══════════════════════════════════════════════════════════════════════════════

async function stakeWithOperator(amount, lockDays, signer) {
    const delegation = new ethers.Contract(CONTRACTS.DELEGATION, DELEGATION_ABI, signer);
    const bkc = new ethers.Contract(CONTRACTS.BKC, BKC_ABI, signer);
    
    const amountWei = ethers.parseEther(amount.toString());
    const ethFee = await delegation.getStakeEthFee(amountWei);
    
    // Approve BKC
    await bkc.approve(CONTRACTS.DELEGATION, amountWei);
    
    // Stake - YOU earn commission!
    const tx = await delegation.stake(
        amountWei,
        lockDays,
        MY_OPERATOR_ADDRESS,  // <-- Your earnings!
        { value: ethFee }
    );
    
    return tx.wait();
}

// ═══════════════════════════════════════════════════════════════════════════════
// GAMING INTERFACE - You earn from every bet
// ═══════════════════════════════════════════════════════════════════════════════

async function playGameWithOperator(wagerAmount, guesses, signer) {
    const fortune = new ethers.Contract(CONTRACTS.FORTUNE, FORTUNE_ABI, signer);
    const bkc = new ethers.Contract(CONTRACTS.BKC, BKC_ABI, signer);
    
    const wagerWei = ethers.parseEther(wagerAmount.toString());
    const ethFee = await fortune.getCommitEthFee();
    
    // Approve BKC
    await bkc.approve(CONTRACTS.FORTUNE, wagerWei);
    
    // Commit play - YOU earn commission!
    const tx = await fortune.commitPlay(
        wagerWei,
        guesses,
        MY_OPERATOR_ADDRESS,  // <-- Your earnings!
        { value: ethFee }
    );
    
    return tx.wait();
}

// ═══════════════════════════════════════════════════════════════════════════════
// NOTARY INTERFACE - You earn from every certification
// ═══════════════════════════════════════════════════════════════════════════════

async function notarizeWithOperator(documentHash, signer) {
    const notary = new ethers.Contract(CONTRACTS.NOTARY, NOTARY_ABI, signer);
    const bkc = new ethers.Contract(CONTRACTS.BKC, BKC_ABI, signer);
    
    const ethFee = await notary.getEthFee();
    
    // Approve BKC for notary fee
    await bkc.approve(CONTRACTS.NOTARY, ethers.parseEther('100'));
    
    // Notarize - YOU earn commission!
    const tx = await notary.notarize(
        documentHash,
        MY_OPERATOR_ADDRESS,  // <-- Your earnings!
        { value: ethFee }
    );
    
    return tx.wait();
}
```

### React Component Example

```jsx
import { useState } from 'react';
import { ethers } from 'ethers';

// Your operator address - CHANGE THIS TO YOUR WALLET!
const OPERATOR = '0xYourWalletAddress';

function StakingWidget() {
    const [amount, setAmount] = useState('');
    const [days, setDays] = useState(30);
    
    async function handleStake() {
        const provider = new ethers.BrowserProvider(window.ethereum);
        const signer = await provider.getSigner();
        
        const delegation = new ethers.Contract(
            '0x41B1B7940E06318e9b161fc64524FaE7261e8739',
            ['function stake(uint256,uint256,address) payable'],
            signer
        );
        
        const amountWei = ethers.parseEther(amount);
        
        // Every stake through this widget earns YOU commission!
        await delegation.stake(amountWei, days, OPERATOR, {
            value: ethers.parseEther('0.0001')
        });
    }
    
    return (
        <div>
            <h2>Stake BKC</h2>
            <input 
                value={amount} 
                onChange={e => setAmount(e.target.value)}
                placeholder="Amount"
            />
            <select value={days} onChange={e => setDays(e.target.value)}>
                <option value={7}>7 days</option>
                <option value={30}>30 days</option>
                <option value={90}>90 days</option>
                <option value={365}>365 days</option>
            </select>
            <button onClick={handleStake}>Stake</button>
        </div>
    );
}
```

## FAQ

### Do I need permission to become an operator?

No. The smart contracts don't know or care who you are. If you pass a valid wallet address as the operator parameter, that address receives commissions. No registration, no approval, no KYC.

### How much can I earn?

There's no limit. Your earnings depend on:
- How many users use your interface
- How much volume flows through your platform
- Which services you integrate (some have higher fees than others)

### Can Backchain revoke my operator status?

No. There is no "operator status" to revoke. The protocol is permissionless. As long as the smart contracts exist on Arbitrum, anyone can be an operator.

### What if someone copies my interface?

Competition is healthy. Build a better product, provide better UX, build a stronger community. The protocol is open to everyone - that's the point.

### Do I need to hold BKC to be an operator?

No. You just need a wallet address. Your users pay the fees, you receive the commissions.

### Can I be an operator for multiple services?

Yes! Build interfaces for all services and earn from all of them. There's no limit.
