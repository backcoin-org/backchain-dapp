## Username System

### Reserve Your Username

Usernames are NFTs that you own. Shorter = more expensive:

| Length | Example | Price Multiplier |
|--------|---------|------------------|
| 1 char | @A | 100x base |
| 2 chars | @AB | 50x base |
| 3 chars | @ABC | 25x base |
| 4 chars | @ABCD | 10x base |
| 5 chars | @ABCDE | 5x base |
| 6+ chars | @ABCDEF | 1x base |

### Username Rules

- Alphanumeric characters only (a-z, 0-9)
- Case insensitive (stored lowercase)
- Cannot be changed once reserved
- Can be transferred/sold like any NFT
- Owner can release username (burns NFT)

## Trust Badge System

### Earn Trust Through Activity

Trust Badges are reputation markers earned through genuine engagement:

| Badge Level | Requirements | Benefits |
|-------------|--------------|----------|
| **Bronze** | 10+ posts, 5+ followers | Basic verification |
| **Silver** | 50+ posts, 25+ followers | Enhanced visibility |
| **Gold** | 200+ posts, 100+ followers | Priority in feeds |
| **Platinum** | 500+ posts, 500+ followers | Premium features |
| **Diamond** | 1000+ posts, 1000+ followers | Maximum credibility |

### Why Trust Badges?

- Combat spam and bots
- Reward genuine participants
- Build reputation over time
- Badges are on-chain and verifiable

## JavaScript Integration (V7)

### Basic Usage

```javascript
import { ethers } from 'ethers';

const BACKCHAT = '0x0D8c2862df03F5be4b569C5ffF4D3aaAEE44BDDb';
const BKC_TOKEN = '0x38DD1898b7B11f07C03e7dBd957fCC8021fB1396';
const MY_OPERATOR_ADDRESS = '0xYourAddress'; // YOUR address to earn commissions!

const backchatABI = [
    "function post(string _content, address _operator) external payable returns (uint256)",
    "function reply(uint256 _parentPostId, string _content, address _operator) external payable returns (uint256)",
    "function like(uint256 _postId, address _operator) external payable",
    "function follow(address _user, address _operator) external payable",
    "function tip(uint256 _postId, uint256 _amount, address _operator) external payable",
    "function reserveUsername(string _username, address _operator) external payable",
    "function getPost(uint256 _postId) view returns (address, string, uint256, uint256, uint256, uint256, uint256, bool)",
    "function getProfile(address _user) view returns (string, uint256, uint256, uint256, uint8, uint256)",
    "function getPostFee() view returns (uint256, uint256)",
    "function getLikeFee() view returns (uint256, uint256)",
    "function getUsernameFee(string _username) view returns (uint256, uint256)",
    "event Posted(uint256 indexed postId, address indexed author, string content, address operator)"
];

const bkcABI = [
    "function approve(address spender, uint256 amount) returns (bool)",
    "function allowance(address owner, address spender) view returns (uint256)"
];
```

### Create a Post

```javascript
async function createPost(content, signer) {
    const backchat = new ethers.Contract(BACKCHAT, backchatABI, signer);
    const bkc = new ethers.Contract(BKC_TOKEN, bkcABI, signer);
    
    // Get fees
    const [bkcFee, ethFee] = await backchat.getPostFee();
    
    console.log('═══════════════════════════════════════════════');
    console.log('CREATE POST');
    console.log('═══════════════════════════════════════════════');
    console.log('Content:', content);
    console.log('BKC Fee:', ethers.formatEther(bkcFee), 'BKC');
    console.log('ETH Fee:', ethers.formatEther(ethFee), 'ETH');
    
    // Approve BKC if needed
    const allowance = await bkc.allowance(await signer.getAddress(), BACKCHAT);
    if (allowance < bkcFee) {
        await bkc.approve(BACKCHAT, ethers.MaxUint256);
    }
    
    // Post (pass YOUR address as operator!)
    const tx = await backchat.post(content, MY_OPERATOR_ADDRESS, { value: ethFee });
    const receipt = await tx.wait();
    
    // Parse event to get postId
    const iface = new ethers.Interface(backchatABI);
    for (const log of receipt.logs) {
        try {
            const parsed = iface.parseLog(log);
            if (parsed?.name === 'Posted') {
                console.log('Post ID:', parsed.args.postId.toString());
                console.log('Posted successfully!');
                return parsed.args.postId;
            }
        } catch {}
    }
}
```
